# OmniSense Command Center

This repository contains the OmniSense Command Center frontend application.

## Quickstart

Requirements: Node.js and npm installed.

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

The app will be available at http://localhost:5173 (default Vite port) unless otherwise configured.

## Technologies

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

## Deploy

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
